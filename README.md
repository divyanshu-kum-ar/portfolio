# Divyanshu Kumar - Portfolio

A modern, responsive, and performance-focused portfolio website showcasing my projects, technical skills, achievements, and experience as an aspiring Software Engineer and MERN Stack Developer.

## 🚀 About

This portfolio highlights my journey in software development, featuring projects built with modern web technologies, problem-solving experience, and a strong foundation in full-stack development.

## ✨ Features

* Responsive design for all devices
* Smooth animations with Framer Motion
* Project showcase with live demos and source code
* Skills and technologies section
* Experience and achievements timeline
* Contact form for professional inquiries
* Resume access
* Social media integration

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* JavaScript (ES6+)
* Tailwind CSS
* Framer Motion

### Tools & Platforms

* Git & GitHub
* VS Code
* Vercel
* MongoDB
* Node.js
* Express.js

## ⚙️ Installation

Clone the repository:

```bash
git clone 
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
